title: How to add media queries in Antd
date: '2022-07-08'
description: 'Here is how I added media queries to Antd components'
categories: ['antd']
---

```js
const layout = {
  labelCol: {
    xs: { span: 24 },
    sm: { span: 12 },
    md: { span: 8 },
    lg: { span: 8 },
  },
  wrapperCol: {
    xs: { span: 24 },
    sm: { span: 12 },
    md: { span: 12 },
    lg: { span: 12 },
  },
};
```

## Add the layout via spread operator onto the Antd component

```js
<Form {...layout}>content</Form>
```
