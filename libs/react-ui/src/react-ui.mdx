`Control`
Components that the user interacts with to control the application's behavior.

- Button
- Checkbox
- Radio
- Select
- Slider
- Switch
- Date Picker
- Time Picker
- Rating
- Multi-step Form

`Display`
Components used to display content or data.

- Card
- Badge
- Hero
- Text
- List
- ListItem
- Table
- Tooltip
- Popover
- Alert
- Label
- Avatar
- Status Indicator

`Feedback`
Components that provide feedback to the user, such as loading states or notifications.

- Spinner
- Modal
- Snackbar
- Toast
- Dialog
- Progress
- Loading Placeholder (Skeleton)

`Form`
Components related to forms and user input.

- Form
- Input
- TextArea
- Checkbox
- Radio
- Select
- Switch
- Slider
- Date Picker
- Time Picker
- Input With Feedback (error, warning, success messages)
- Autocomplete

`Layout`
Components used to construct the layout of a page.

- Box
- Grid
- Flex
- Drawer
- Container
- Header
- Footer
- Sidebar
- Divider
- Space
- Layout

`Media`
Components used to display media such as images, video, and audio.

- Image
- Video
- Audio
- Carousel
- Gallery
- Lightbox

`Navigation`
Components that assist users in navigating the application.

- Navbar
- Breadcrumb
- Pagination
- Tabs
- List
- Select
- Steps (or Wizard)
- Link

`Typography`
Components related to text and font display.

- Heading
- Subheading
- Text
- Caption
- Quote

`Utility`
Components that provide certain functionality but might not fit directly into the other categories.

- Tooltip
- Popover
- Collapse
- Accordion
- SearchBar
- Stepper
- Icon
- SEO

`Pages`
Components that represent full pages or large parts of pages.

- About
- Contact
- Fax
- PageNotFound
- PaymentFailed
- PaymentSucceeded
- PrivacyPolicy
- TermsAndConditions
